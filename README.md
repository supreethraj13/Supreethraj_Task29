# React Fetch Hook Example

This is a simple Vite React app that shows how to use a custom hook to fetch data from an API.

The hook is called `useFetch` and it returns:
- `data` — the response data
- `loading` — a boolean while the request is in progress
- `error` — any errors from the fetch request

## What is included

- `src/hooks/useFetch.jsx` — custom hook
- `src/App.jsx` — photo gallery page using the hook
- `src/main.jsx` — app entry point
- `src/App.css` — page styling

## How to run

```bash
cd react/Hoocks_task
npm install
npm run dev
```

## Notes

- The app fetches photo data from `jsonplaceholder.typicode.com`.
- The demo shows a gallery of photo thumbnails.
- It is a simple example, not a finished production app.

## Build

```bash
npm run build
```
